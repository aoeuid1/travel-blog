# **App Name**: WanderLogger

## Core Features:

- Create Post: Create a new travel blog post with title, content, and embedded photos.
- List Entries: View a chronological list of all blog posts with brief previews and dates.
- Detail View: Display a detailed individual post page with full content and embedded photos.
- AI Content Assistant Tool: AI tool that reviews travel blog entries to suggest related points of interest near places mentioned.

## Style Guidelines:

- Primary color: Soft coral (#F08080) to evoke a sense of warmth and exploration.
- Background color: Very light desaturated coral (#F8E8E8) to keep focus on content.
- Accent color: Muted teal (#70A1A1) to provide contrast and highlight interactive elements.
- Body font: 'Literata', a serif font to enhance readability and give a classic feel.
- Headline font: 'Belleza', a sans-serif to add personality and style to the headlines.  Body text should still use Literata.
- Use outline icons related to travel, such as planes, maps, and cameras.
- Clean, responsive layout that adapts to different screen sizes for optimal viewing experience.
- Subtle animations when navigating between pages and loading content.
